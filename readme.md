# 🌱 Projetos Git

Repositório dedicado à prática e ao estudo de **comandos e fluxos de trabalho do Git**, criado como parte do meu aprendizado como estudante de Engenharia de Software.

## 🎯 Objetivo

Este projeto serve como um "laboratório" pessoal para experimentar comandos do Git na prática, entender como o controle de versão funciona no dia a dia de um desenvolvedor e consolidar conceitos estudados em aula.

## 🛠️ Conceitos praticados aqui

- [x] Inicialização de repositório (`git init`)
- [x] Commits e histórico (`git commit`, `git log`)
- [x] Edição e versionamento de arquivos
- [ ] Branches e merge (`git branch`, `git merge`)
- [ ] Resolução de conflitos
- [ ] Trabalho com repositórios remotos (`git push`, `git pull`, `git fetch`)

> Vou marcando os tópicos conforme vou praticando cada um.
